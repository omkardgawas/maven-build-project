\# Maven Build Automation Project



\## Objective

Automate the build process of a Java application using Maven.



\## Tools Used

\- Java

\- Maven



\## Steps

1\. Created Maven project using archetype

2\. Managed dependencies using pom.xml

3\. Built project using mvn clean package

4\. Generated JAR file



\## Output

Successfully built Java application using Maven.



\## Run Commands

mvn clean package

java -cp target/maven-app-1.0-SNAPSHOT.jar com.devops.project.App
